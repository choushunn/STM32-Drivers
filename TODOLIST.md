# STM32 常用驱动开发清单

记录项目中必须掌握的常用嵌入式驱动，按子系统分类。勾选表示已完成。

---

## 显示类

- [x] **OLED (SH1106 / SSD1306)** — I2C/SPI 驱动, 绘图/字符/数值显示
- [ ] **LCD 1602 (I2C 转接板)** — PCF8574 转 I2C, 字符显示
- [ ] **TFT LCD (SPI, ILI9341 等)** — 彩屏驱动, 支持 BMP/JPEG 显示
- [ ] **segment LED (TM1650 / MAX7219)** — 数码管驱动, 动态扫描

## 传感器类

- [x] **DHT11** — 单总线温湿度传感器, 校验和检测
- [x] **HC-SR04** — 超声波测距, EMA 滤波 + 溢出补偿
- [ ] **DS18B20** — 单总线高精度温度传感器 (-55°C ~ +125°C)
- [ ] **MPU6050** — I2C 六轴陀螺仪 + 加速度计, DMP 姿态解算
- [ ] **BMP180 / BMP280** — I2C 气压 + 温度传感器, 海拔换算
- [ ] **VL53L0X** — I2C 激光测距 (ToF, 最大 2m)
- [ ] **ACS712** — 模拟霍尔电流传感器, ADC 采样 + 滤波

## 输入类

- [x] **Joystick (模拟摇杆)** — 双轴 ADC + 按键, 方向判别 + 死区
- [x] **Rotary Encoder (旋转编码器)** — 正交解码 + 按键消抖 + 长短按
- [ ] **矩阵键盘 (4x4)** — 行列扫描 + 消抖 + 组合键
- [ ] **触摸按键 (TTP224)** — 四通道电容触摸, GPIO 电平读取
- [ ] **PS2 手柄** — SPI 协议, 摇杆 / 按键 / 振动反馈

## 通信类

- [ ] **ESP8266 / ESP-01S (AT 指令)** — UART Wi-Fi, TCP/IP 透传
- [ ] **HC-05 / HC-06 蓝牙模块** — UART 蓝牙透传, AT 指令配置
- [ ] **NRF24L01** — SPI 2.4G 无线, 一对多通信
- [ ] **LoRa (SX1278)** — SPI 远距离低功耗无线通信
- [ ] **CAN (TJA1050)** — CAN 总线收发, 多节点通信
- [ ] **RS485 (MAX485)** — 半双工差分总线, Modbus 协议

## 存储类

- [ ] **AT24Cxx (EEPROM)** — I2C 非易失存储, 页写入
- [ ] **W25Qxx (SPI Flash)** — SPI Flash 读写, FATFS 文件系统
- [ ] **SD Card (SPI)** — SPI SD 卡读写, FATFS 文件系统

## 执行器类

- [x] **SG90 / MG996R 舵机** — PWM 驱动, 角度控制, 平滑转动
- [ ] **L298N / L9110S 电机驱动** — GPIO + PWM, 正反转 + 调速
- [ ] **A4988 / DRV8825 步进电机** — STEP/DIR 脉冲控制, 加减速
- [ ] **继电器模块** — GPIO 控制, 光耦隔离
- [ ] **蜂鸣器 (有源 / 无源)** — GPIO / PWM 音乐播放

## 数模转换类

- [x] **DAC1220** — 3线SDIO 20位DAC, 自校准, ±10V输出

## 电源 / 管理类

- [ ] **INA219** — I2C 电压 / 电流 / 功率监测
- [ ] **DS3231 / DS1307** — I2C 实时时钟 (RTC), 备用电池
- [ ] **红外遥控 (NEC 协议)** — 外部中断 + 定时器捕获, 按键解码
